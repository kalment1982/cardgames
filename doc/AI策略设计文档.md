# 拖拉机游戏AI策略设计文档

## 文档概述

本文档基于对拖拉机（升级、80分）游戏的深入研究，整合了多个来源的游戏技巧和策略，为AI玩家设计了一套完整的决策系统。

## 游戏基础知识

### 游戏目标
- 庄家方：保底（防止闲家得到80分或更多）
- 闲家方：抓分（获得80分或更多以升级）
- 分牌价值：K=10分，10=10分，5=5分，全局总分200分

### 术语说明
- **庄家方**：坐庄玩家及其队友（2人）
- **闲家方**：非庄家方的另一队（2人）
- **坐庄玩家**：当前拥有庄家特权的玩家（拿底牌、扣底、首墩先出）
- 本文档中"庄家"如无特别说明，指庄家方（2人）

### 牌型结构
1. **单张**：一张牌
2. **对子**：两张相同的牌
3. **拖拉机**：两个或更多连续的对子（如：33 44 或 77 88 99）
   - **断档拖**：当级牌在中间时，级牌两侧的对子可构成拖拉机
   - 示例：级牌为8时，77+99算拖拉机（因为8是级牌，7和9在主牌序列中相邻）
4. **甩牌**：同花色多张牌的组合出牌

### 主牌体系
- 大王 > 小王 > 主级牌（主花色） > 副级牌（其他花色，彼此无大小区别） > 主花色其他牌
- 副牌按花色分类，同花色内按点数大小排序
- 注意：副级牌之间无大小区别，但都大于主花色A

## 核心策略原则

### 1. 庄家策略

#### 1.1 底牌埋牌策略（坐庄玩家专属）
**目标**：保护分牌，控制牌权

**前置条件**：
- 坐庄玩家拿到底牌后，手牌共33张
- 需要从33张中选择8张扣底，保持手牌25张

**策略要点**：
- **优先埋小副牌**：埋没有分值的小副牌（2、3、4、6、7、8、9、J、Q、A）
- **避免埋分牌**：尽量不埋K、10、5，除非该花色特别短且无法保护
- **避免埋主牌**：主牌是控制权的关键，不应轻易埋掉
- **花色平衡**：如果某个副牌花色特别短（1-2张），可以考虑全部埋掉，形成"缺门"优势
- **埋牌组合**：如果必须埋分牌，优先埋单张分牌而非对子分牌
- **抠底风险**：考虑被闲家抠底的风险，避免埋入过多分牌

**实现建议**：
```
优先级排序：
1. 单张小副牌（非分牌）
2. 短门副牌（2-3张的花色）
3. 对子小副牌
4. 最后才考虑分牌或主牌
```

#### 1.2 首家出牌策略
**目标**：控制节奏，保护底牌

**策略要点**：
- **试探性出牌**：前期用副牌试探闲家的牌型分布
- **主牌控制**：保留主牌大牌（大小王、主级牌）用于关键时刻
- **长套优势**：如果有某个花色特别长（6张以上），可以考虑出长套消耗对手
- **拖拉机优先**：有拖拉机时优先出，既能多出牌又能控制
- **分牌保护**：避免在前期出带分的牌型，除非确信能赢

**出牌顺序建议**：
1. 短门副牌（3-4张）- 快速清理
2. 长套副牌拖拉机 - 建立优势
3. 主牌小牌 - 试探
4. 保留主牌大牌和分牌到后期

#### 1.3 跟牌策略（庄家方）
**目标**：尽量赢牌，避免送分

**跟牌硬约束**（必须遵守）：
- 必须跟与首引相同张数
- 有首引花色必须先跟该花色
- 首引含对子/拖拉机时，若手牌中可组成对应牌型，必须优先跟对应牌型
- 无首引花色时可垫牌或上主毙牌

**策略要点**：
- **有同花色**：
  - 如果对家（庄家队友）已经赢牌：在满足硬约束前提下，出小牌保留大牌
  - 如果闲家赢牌且有分：在满足硬约束前提下，用大牌争取赢回
  - 如果无分：在满足硬约束前提下，出中等牌

- **缺门情况**：
  - 优先用主牌毙牌（特别是闲家出分牌时）
  - 如果主牌不够大，出小主牌或垃圾副牌
  - 绝不在缺门时送出分牌

**跟牌决策流程**：
```
1. 判断是否有首引花色
   - 有 → 必须先跟该花色
   - 无 → 可垫牌或毙牌

2. 判断是否需要跟对应牌型
   - 首引对子且手中有对子 → 必须跟对子
   - 首引拖拉机且手中有拖拉机 → 必须跟拖拉机

3. 在满足硬约束前提下，根据局势选择牌的大小
   - 对家赢牌 → 出小牌
   - 闲家赢牌有分 → 出大牌争胜
   - 无分 → 出中等牌
```

### 2. 闲家策略

#### 2.1 首家出牌策略
**目标**：抓分，破坏庄家节奏

**策略要点**：
- **分牌优先**：有把握赢的情况下，优先出带分的牌型
- **长套攻击**：用长套副牌消耗庄家的主牌
- **拖拉机进攻**：拖拉机是强力武器，可以迫使庄家用大牌
- **试探出牌**：前期可以用小牌试探庄家的牌型
- **配合对家**：注意对家的信号，协调进攻

**出牌策略**：
1. **前期（前5轮）**：
   - 试探性出牌，了解各家牌型
   - 出短门副牌，寻找庄家弱点

2. **中期（6-10轮）**：
   - 主动进攻，出长套或拖拉机
   - 配合对家，形成连续攻击

3. **后期（11轮以后）**：
   - 集中火力抓分
   - 利用主牌优势强行得分

#### 2.2 跟牌策略（闲家方）
**目标**：配合对家，争取得分

**跟牌硬约束**（必须遵守）：
- 必须跟与首引相同张数
- 有首引花色必须先跟该花色
- 首引含对子/拖拉机时，若手牌中可组成对应牌型，必须优先跟对应牌型
- 无首引花色时可垫牌或上主毙牌

**策略要点**：
- **对家赢牌**：
  - 对家已经赢牌：在满足硬约束前提下，尽量选择分牌（K、10、5）
  - 对家出分牌：在满足硬约束前提下，尽量不要盖过对家的牌

- **庄家赢牌**：
  - 有大牌：在满足硬约束前提下，争取赢回牌权
  - 无大牌：在满足硬约束前提下，出小牌保留实力，不送分

- **缺门情况**：
  - 对家赢牌：优先选择分牌垫出
  - 庄家赢牌：选择垃圾牌垫出，保留主牌和分牌

**跟牌决策流程**：
```
1. 判断是否有首引花色
   - 有 → 必须先跟该花色
   - 无 → 可垫牌或毙牌

2. 判断是否需要跟对应牌型
   - 首引对子且手中有对子 → 必须跟对子
   - 首引拖拉机且手中有拖拉机 → 必须跟拖拉机

3. 在满足硬约束前提下，根据局势选择牌的大小和类型
   - 对家赢牌 → 优先选分牌
   - 庄家赢牌有大牌能赢 → 出大牌争胜
   - 庄家赢牌无法赢 → 出小牌保留
```

### 3. 对家配合策略

#### 3.1 信号牌系统
**目标**：通过出牌传递信息

**常见信号**：
1. **首家出单张大牌**：表示该花色有实力，希望对家配合
2. **首家出单张小牌**：表示该花色较弱，试探性出牌
3. **跟牌出大牌**：表示该花色有控制力
4. **跟牌出小牌**：表示该花色较弱或保留实力

#### 3.2 配合时机
**关键场景**：
1. **连续出牌**：对家赢牌后，下一轮继续配合该花色
2. **分牌保护**：对家出分牌时，尽量帮助对家赢牌
3. **主牌配合**：对家用主牌毙牌时，下轮可以继续该花色
4. **底牌保护**：后期注意保护底牌，避免被闲家抓底

### 4. 高级技巧

#### 4.1 记牌技巧
**目标**：推断对手牌型

**记牌要点**：
1. **主牌数量**：记录已出的主牌，推断剩余主牌分布
2. **分牌位置**：记录K、10、5的位置和数量
3. **花色长度**：根据出牌推断各家花色长度
4. **大牌位置**：记录大王、小王、主级牌的位置

**推断方法**：
- 某家缺门：该花色已出完
- 某家连续出小牌：可能保留大牌
- 某家用主牌毙牌：该副牌花色已缺门

#### 4.2 算牌技巧
**目标**：计算得分和剩余牌

**计算要点**：
1. **已得分数**：实时计算当前得分
2. **剩余分数**：计算场上剩余分牌
3. **底牌倍数**：后期注意底牌的倍数效应
4. **升级计算**：根据得分判断能否升级

**关键数字**：
- 总分：200分
- 升级线：80分（闲家方）
- 保底线：80分（庄家方）
- 底牌：8张

**抠底规则**：
- 闲家方拿下最后一墩时触发抠底
- 庄家方拿下最后一墩时，底牌分数计入庄家方，但不翻倍
- 抠底倍数按最后一墩牌型计算：
  - 单张抠底：底牌分数 × 2
  - 对子抠底：底牌分数 × 4
  - 拖拉机抠底：底牌分数 × 2^n（n=连对数量）
  - 示例：3对拖拉机抠底，倍数为2^3=8倍

#### 4.3 甩牌技巧
**目标**：一次性出多张牌

**甩牌条件**：
- 同花色多张牌
- 对手无法全部跟上
- 可以包含单张、对子、拖拉机的组合

**甩牌策略**：
1. **长套甩牌**：有6张以上同花色，可以尝试甩牌
2. **分牌甩牌**：甩牌中包含分牌，增加得分
3. **清理手牌**：后期用甩牌快速清理手牌
4. **风险控制**：甩牌失败会被拆分，需谨慎使用

**甩牌时机**：
- 前期：不建议甩牌，容易被拆
- 中期：有明显长套优势时可以尝试
- 后期：手牌较少时，甩牌成功率高

#### 4.4 拖拉机运用
**目标**：最大化拖拉机的价值

**拖拉机优势**：
1. 一次出多张牌，加快节奏
2. 对手必须用拖拉机跟，消耗对手资源
3. 容易建立牌权优势

**拖拉机策略**：
- **进攻型**：用副牌拖拉机消耗对手主牌
- **防守型**：用主牌拖拉机控制局面
- **得分型**：拖拉机中包含分牌，增加得分

**拖拉机时机**：
1. 有明显拖拉机优势（3对以上连续）
2. 需要快速建立牌权
3. 对手主牌较弱时

### 5. 难度等级设计

#### 5.1 简单难度（Level 1-2）
**特点**：基础AI，适合新手

**策略简化**：
- 随机性较高（30-40%随机决策）
- 基本跟牌规则
- 简单的出牌优先级
- 不考虑对家配合
- 不记牌

**实现要点**：
```
- 首家出牌：随机选择合法牌型
- 跟牌：优先跟同花色，缺门随机出牌
- 埋牌：随机选择8张小牌
```

#### 5.2 中等难度（Level 3-5）
**特点**：有一定策略，适合进阶玩家

**策略要点**：
- 随机性中等（15-25%随机决策）
- 基本的庄闲策略区分
- 简单的对家配合
- 基础记牌（主牌和分牌）
- 考虑牌型优先级

**实现要点**：
```
- 首家出牌：优先拖拉机和对子
- 跟牌：区分庄闲，基本配合
- 埋牌：避免埋分牌和主牌
- 记牌：记录主牌和分牌数量
```

#### 5.3 困难难度（Level 6-8）
**特点**：高级AI，适合高手

**策略要点**：
- 随机性较低（5-10%随机决策）
- 完整的庄闲策略
- 良好的对家配合
- 详细记牌和算牌
- 考虑多步决策

**实现要点**：
```
- 首家出牌：根据局势选择最优策略
- 跟牌：精确配合，争取最大收益
- 埋牌：战略性埋牌，形成优势
- 记牌：详细记录所有出牌
- 算牌：计算得分和剩余牌
```

#### 5.4 专家难度（Level 9-10）
**特点**：顶级AI，接近人类高手

**策略要点**：
- 随机性极低（0-5%随机决策）
- 深度策略分析
- 完美对家配合
- 完整记牌和算牌系统
- 多步预判和博弈

**实现要点**：
```
- 首家出牌：多步预判，选择最优路径
- 跟牌：完美配合，最大化收益
- 埋牌：战略性埋牌，考虑多种情况
- 记牌：完整记录所有信息
- 算牌：精确计算概率和期望
- 博弈：考虑对手可能的策略
```

## AI决策流程

### 0. 亮主决策流程（发牌阶段）
```
输入：当前手牌，当前亮主状态，玩家角色
输出：是否亮主，亮主牌型

步骤：
1. 判断是否有级牌
   - 无级牌 → 不亮主
   - 有级牌 → 继续评估

2. 评估亮主价值
   - 统计该花色的牌力
   - 评估主牌优势
   - 考虑对手可能的反主

3. 判断是否反主
   - 当前已有人亮主 → 评估是否反主
   - 反主条件：有对级牌且该花色牌力更强

4. 特殊情况：坐庄玩家自保
   - 上局主花色牌力强 → 考虑自保

5. 返回决策
   - 亮主/反主/不亮主
```

### 1. 埋底牌决策流程（坐庄玩家专属）
```
输入：手牌（33张，包含底牌）
输出：埋牌（8张）

步骤：
1. 分析手牌结构
   - 统计各花色数量
   - 识别主牌和副牌
   - 标记分牌位置

2. 评估埋牌候选
   - 单张小副牌（优先级最高）
   - 短门副牌（2-3张）
   - 对子小副牌
   - 分牌（最后考虑）

3. 选择最优组合
   - 避免埋主牌
   - 避免埋分牌
   - 保持手牌平衡
   - 考虑抠底风险

4. 返回埋牌列表（8张）
   - 埋牌后手牌剩余25张
```

### 2. 首家出牌决策流程
```
输入：手牌，当前局势
输出：出牌列表

步骤：
1. 生成候选牌型
   - 单张候选
   - 对子候选
   - 拖拉机候选
   - 甩牌候选

2. 评估每个候选
   - 牌型强度
   - 得分潜力
   - 风险评估
   - 后续影响

3. 根据角色选择策略
   - 庄家：控制为主
   - 闲家：进攻为主

4. 选择最优候选
   - 综合评分
   - 考虑随机性（根据难度）

5. 返回出牌列表
```

### 3. 跟牌决策流程
```
输入：手牌，首家出牌，当前牌局
输出：跟牌列表

步骤：
1. 判断是否有同花色
   - 有：生成同花色候选
   - 无：生成缺门候选

2. 判断当前谁赢牌
   - 对家赢牌
   - 对手赢牌
   - 自己可能赢牌

3. 根据角色和局势选择策略
   - 庄家跟牌策略
   - 闲家跟牌策略
   - 对家配合策略

4. 生成候选跟牌
   - 大牌候选（争胜）
   - 小牌候选（保留）
   - 分牌候选（送分）
   - 垃圾牌候选（弃牌）

5. 评估并选择最优候选
   - 考虑得分
   - 考虑牌权
   - 考虑后续

6. 返回跟牌列表
```

## 评估指标

### 1. AI性能指标
- **胜率**：庄家方保底率，闲家方升级率
- **得分效率**：平均每局得分
- **决策时间**：每手决策时间 ≤ 1.5秒（硬性要求）
- **配合度**：对家配合成功率

### 2. 策略有效性指标
- **主牌控制率**：主牌赢牌次数/总出牌次数
- **分牌获取率**：获得分牌数量/总分牌数量
- **拖拉机成功率**：拖拉机赢牌次数/拖拉机出牌次数
- **甩牌成功率**：甩牌成功次数/甩牌尝试次数
