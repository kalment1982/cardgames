## 3. 冻结术语

### 3.1 阶段口径
- 全局“前期” = 前 3 墩
- `Lead-001` 的强优先窗口 = 前 2 墩
- 第 3 墩开始，若前两墩的稳副牌路线仍然有效，可以延续，但不再享受开局优先级加成

### 3.2 绝门判断口径
- `确认绝门（100%）`：已观察到该玩家跟牌时未能跟出该花色，而是垫其他副牌或上主
- `大概率未绝门（>= 70% 置信度）`：基于记牌与剩余牌估计，该玩家在该花色仍有未出牌的概率 `>= 70%`

### 3.3 底牌分分档
- 低底分：`<= 10`
- 中底分：`11 ~ 29`
- 高底分：`>= 30`

## 4. 首发阶段冻结口径

### 4.1 首发总优先级顺序
首发阶段统一按以下顺序处理：

1. 当前高收益安全甩牌
2. 庄家稳副兑现 / 稳副协作跑分 / 低分安全甩牌（同级比较后续收益）
3. 有明确收益的自己抽主
4. 明确把牌权交给队友
5. 主动经营未来甩牌
6. 主动做绝门
7. 低价值信息试探

### 4.2 安全甩牌优先级口径
- 只有当本次甩牌预期得分 `>= 10` 时，安全甩牌才排在首发策略最前
- 若本次甩牌预期得分 `< 10`，尤其是 `0` 分安全甩牌，则降到和“稳副兑现 / 稳副协作跑分”同级，按后续收益更高者选择
- 说明：
  - `>= 10` 是“当前直接兑现收益”的门槛
  - `>= 15` 才允许“为甩牌调主”，是因为调主会额外消耗主牌资源，代价高于直接甩牌

### 4.3 `Lead-001` 庄家前两墩优先跑稳副牌
冻结口径：
- 强优先窗口仅限前 2 墩
- 前 2 墩若已经明显失去该门主导权，则立即退出该策略
- 第 3 墩若仍继续跑该门，只能因为“后续收益仍高”，不再享受开局优先级加成

### 4.4 `Lead-003` 抽主必须有明确收益
冻结口径：
- 抽主只能用于：
  - 压缩对手毙牌空间
  - 为本方跑副牌或保底铺路
  - 争取末盘最后控制
- 如果“出主牌”的真实目的是交给队友接牌，归 `Lead-007`
- 如果“出主牌”的真实目的是为未来甩牌削弱对手主结构，归 `Lead-008`

### 4.5 `Lead-005` 与 `Lead-008` 的边界
冻结口径：
- 当前已经达到高收益安全甩牌，直接执行 `Lead-005`
- `Lead-008` 只负责经营未来甩牌，不与当前可直接甩牌平行竞争

### 4.6 `Lead-006` 稳副协作跑分
冻结口径：
- `Lead-006` 是 `Lead-001` 的协作增强版，不是平行独立路线
- 继续跑稳副牌时，后位关键对手满足“大概率未绝门（>= 70%）”即可，不要求 100% 确认

### 4.7 `Lead-007` 交牌权给队友
冻结口径：
- 必须同时满足：
  - 自己没有明显后续线
  - 队友有正面接手证据

定义：
- “自己没有明显后续线” = 下一手没有稳副可跑、没有安全甩牌、没有有收益的抽主
- “队友有正面接手证据” = 至少满足以下之一：
  - 已观察到队友某门强，出过大牌且该门未被毙
  - 已确认队友某门绝门，可喂牌让其毙起
  - 记牌推断显示队友主牌明显厚

硬约束：
- 禁止盲交
- “我没什么好出的，随便交一张看看”不属于 `Lead-007`，而应回退到 `Lead-004` 的低价值信息试探

### 4.8 `Lead-008` 三对子经营甩牌
冻结口径：
- 默认大对子优先出，除非不满足控场条件

“大概率能控场”的定义：
- 该对子是当前该花色已知最大的未出对子；或
- 该对子为 `AA / KK`，且无证据表明对手持有同花色拖拉机

附加规则：
- `QQ` 及以下对子，除非记牌确认是该花色最大未出对子，否则默认不算“能控场”
- 如果大对子不满足控场条件，则切换到小对子优先
- 小对子优先仅适用于不带分小对子去耗对手对子资源
- 临时口径：
  - 若同时存在两门及以上都可出对子经营，先按“控场确定性”排序
  - 若控场确定性相近，优先出含分更多的门
  - 若含分也接近，优先出更短的门，为后续做绝门或继续对子经营创造空间
  - 该条仅作为当前冻结期的临时口径，后续由“多副牌门优先级”专项细化

### 4.9 `Lead-008` 为甩牌调主的边界
冻结口径：
- 中等主牌也允许为甩牌调主，但必须同时满足以下硬约束：
  - 调主后手中仍保留 `>= 2` 张主牌，且其中至少 `1` 张是 `大王 / 小王 / 主牌A级别的控制牌`；或保留 `>= 1` 对主牌对子
  - 预期甩牌收益 `>= 15`
  - 当前不处于保底模式
- 若不满足以上任一条，不允许为甩牌调主

### 4.10 `Lead-009` 做绝门的代价边界
冻结口径：
- 允许拆的：`2~9` 的非分牌小对子
- 不允许拆的：
  - 拖拉机
  - 含分对子（`55 / 1010 / KK`）
  - `A` 对
  - `Q` 及以上对子

附加条件：
- 即使允许拆低价值弱对，也必须满足“绝门后有明确后续收益预期”，例如：
  - 毙高分墩
  - 接队友桥接牌
  - 末盘争控制
  - 闲家争底 / 庄家保底
- 禁止为了绝门而绝门

## 5. 保底 / 抠底冻结口径

说明：
- 本节只冻结 `Bottom-001 / 003 / 006 / 008 / 009 / 011`
- 其余 `Bottom` 编号暂不冻结，不是遗漏

### 5.1 闲家抠底优先级阈值
- 闲家已得分 `>= 50`：开始关注抠底可能性，纳入决策考量
- 闲家已得分 `>= 60`：抠底优先级显著上升，允许为抠底保留主牌结构
- `闲家已得分 + 底牌估计分 × 扣底倍率 >= 80`：进入强抠底模式
- 底牌估计分默认口径：
  - 无明确信号时，默认按 `10` 分估计
  - 出现 `Bottom-009` 的高分信号后，再上调估计值
  - 闲家只能把该值当作估计，不能当作确定事实

### 5.2 庄家模式切换阈值
- `正常运营 -> 保底关注`：
  - `闲家已得分 + 场上剩余可争分 > 50`，且 `底牌分 >= 15`
- `保底关注 -> 强保底`：
  - `闲家已得分 + 底牌分 × 2 >= 70`
- 强保底模式下：
  - 主牌资源默认冻结用于末盘
  - 中盘允许放弃 `<= 10` 分的小分墩
- 定义：
  - `场上剩余可争分` = 所有未出的分牌 + 底牌估计分
  - 庄家视角可直接使用真实底牌分；闲家与公共估计视角使用保守上界
  - 保底判断这里采用保守上界，宁可提前触发，不做乐观估计

### 5.3 `Bottom-001` 庄家保底提前触发
冻结口径：
- 庄家可见底牌且达到“中底分”或更高时，应显著提前进入保底判断
- 达到“高底分”时，保底意图应明显上提，优先级可覆盖普通抢分

### 5.4 `Bottom-003` 单扣 vs 双扣
冻结口径：
- 先判断单扣是否足够赢
- 如果单扣不够，但双扣够，则允许保留对子/拖拉机结构争取更高倍率
- 如果单扣已足够赢，则默认降风险
- 只有当更高倍率候选与单扣候选同样稳时，才允许继续追求双扣

### 5.5 `Bottom-006` 大小王顺序例外
冻结口径：
- 默认“先小王，后大王”
- 但若满足以下任一具体信号，则不能机械先出小王：
  - 记牌显示 `大王` 仍未出，且高概率保留在后手对手手里
  - 后手对手仍可能保有能在同张数主牌场景下反压小王的主对或主拖结构
  - 当前小王只能形成“险胜”，而不能形成“稳赢控制”

### 5.6 `Bottom-008` 闲家抠底是动态模式，不是末墩临时决定
冻结口径：
- 前期主牌很强时，可以较早把抠底纳入重要意图
- 中期根据已得分、剩余分、底牌估计分动态抬升或降低抠底优先级
- 后期以“能否过线”和“倍率是否值得”为最高判断标准

### 5.7 `Bottom-009` 底牌估计技巧
冻结口径：
- 若某花色全员已空，但该花色分牌仍未出现，则这些分牌大概率在底牌
- 若多门都出现“全员已空门但分牌未见”，则底牌高分概率显著上升
- 这条估计同时提升抠底优先级和保底优先级

### 5.8 `Bottom-011` 庄家保底是动态模式
冻结口径：
- 庄家不是到最后一墩才开始保底
- 一旦达到模式切换阈值，应从“普通运营”切换为“守底运营”
- 强保底模式下，主牌资源默认冻结用于末盘，不为中盘小分轻易消耗

## 6. 首批冻结的协作与记牌规则

### 6.1 `Mate-001`
- 队友稳大时，优先垫分而不是乱超
- 冻结边界：
  - 只有在“锁赢”或“高置信稳赢”时，才允许纯送分
  - 若只是“暂时领先”或“险胜”，不适用本条，应转入加固思路，而不是机械送分
- 送分边界：
  - 可优先送低风险分牌
  - 不得为了送分而无意义烧掉关键大主、关键对子或关键拖拉机结构

### 6.2 `Mate-003`
- 在无收益差异时，统一按“小牌优先、保大牌优先”
- 冻结边界：
  - “无收益差异”是指：对本墩输赢、牌权流向、末盘价值、保底/抠底价值都没有本质差异
  - 只有在这些维度都近似等价时，才进入“小牌优先”
- 默认回退：
  - 若信息不足，无法确认是否等价，则默认按保守口径处理：小牌、保结构、保主控

### 6.3 `Memory-002`
- 缺门信息必须强驱动后续出牌
- 首发继续跑稳副牌时，允许使用“大概率未绝门（>= 70%）”口径
- 冻结边界：
  - 确认绝门可直接触发强策略切换
  - 概率口径只能用于调高或调低倾向，不能包装成已知事实

### 6.4 `Memory-006`
- 后位反超风险必须按“还有谁没出”来判断
- 冻结边界：
  - 不能只看“我现在能不能压当前赢家”
  - 必须同时看：
    - 后面还有几家未出
    - 未出者是队友还是对手
    - 未见更大主牌 / 更大同花结构还剩多少
  - 只有综合后位风险后，才能把当前候选判为“险胜 / 稳赢 / 锁赢”
