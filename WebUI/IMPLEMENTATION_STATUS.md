## 技术实现细节

### 游戏流程
```
StartNewGame()
  → Game.StartGame() (发牌)
  → GamePhase.Bidding (亮主阶段)
  → BidTrump() (玩家亮主)
  → FinalizeTrump() (确定主花色)
  → GamePhase.Burying (扣底阶段)
  → BuryBottom() (扣底)
  → GamePhase.Playing (出牌阶段)
  → PlayCards() (出牌)
  → AIPlay() (AI自动出牌)
  → GamePhase.Finished (游戏结束)
```

### AI决策
- 使用 `AIPlayer` 类进行决策
- 难度设置为 `Medium`
- AI角色根据位置判定（庄家队 vs 闲家队）
- AI出牌间隔800ms，模拟思考时间

### 状态管理
- 使用 `Game` 类管理游戏状态
- 使用 `GameState` 保存游戏数据
- UI层维护选中卡牌列表
- 通过 `StateHasChanged()` 触发UI更新
