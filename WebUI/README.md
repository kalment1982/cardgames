# 拖拉机游戏 Web版

## 项目说明

这是拖拉机（80分）游戏的Web版本，使用Blazor WebAssembly技术实现。

## 功能特性

### 已实现功能
- ✅ 欢迎页面（开始游戏入口）
- ✅ 游戏主界面（横屏布局）
- ✅ 左上角信息栏（级别、主花色、抓分）
- ✅ 四方位玩家显示（南北东西）
- ✅ 手牌区（自动排序、选中效果）
- ✅ 出牌显示区
- ✅ 操作按钮（出牌、扣底、查看底牌、上一轮牌）
- ✅ 扑克牌图片资源集成
- ✅ 响应式设计（支持不同屏幕尺寸）

### 待完善功能
- ⏳ 发牌动画
- ⏳ 亮主功能（亮牌框交互）
- ⏳ 扣底功能
- ⏳ 智能选择（对子、拖拉机）
- ⏳ 拖拽出牌
- ⏳ AI自动出牌
- ⏳ 游戏结算
- ⏳ 音效

## 运行方法

### 前置要求
- .NET 6.0 SDK

### 启动步骤

1. 进入WebUI目录：
```bash
cd WebUI
```

2. 运行应用：
```bash
dotnet run
```

3. 打开浏览器访问：
```
http://localhost:5000
```

### 开发模式（热重载）
```bash
dotnet watch run
```

## 项目结构

```
WebUI/
├── Pages/
│   ├── Index.razor          # 欢迎页面

│   └── assets/

```

## UI设计说明

### 布局
- 采用横屏模式（Landscape）
- 目标设备：iPhone 15 Pro Max、华为 Nova 14
- 基准分辨率：2400 × 1080

### 色彩方案
- 背景：深绿色 #0D5C2F（模拟牌桌）
- 选中：黄色 #F1C40F
- 赢墩：绿色 #2ECC71
- 我方：红色 #E74C3C
- 对方：蓝色 #3498DB

### 卡牌尺寸
- 玩家手牌：60px × 90px
- AI手牌：40px × 60px（背面）
- 出牌区：80px × 120px

## 技术栈

- **前端框架**：Blazor WebAssembly
- **UI样式**：CSS3
- **游戏逻辑**：C# (.NET 6.0)
- **图片资源**：PNG格式扑克牌

## 注意事项

1. **横屏模式**：游戏设计为横屏模式，建议在横屏状态下游戏
2. **浏览器兼容性**：推荐使用Chrome、Safari、Edge等现代浏览器
3. **性能优化**：首次加载可能需要几秒钟下载WebAssembly资源
