# 毙主专项测试用例：首发 2单牌 + 1拖拉机 + 1对子

## 目的

验证在“首发为混合多张牌型（2单牌 + 1拖拉机 + 1对子）”且后手存在主牌毙牌的情况下：

- `FollowValidator` 是否接受合法跟牌
- `TrickJudge` 是否正确判定一墩赢家

本专项不验证甩牌是否成功，只验证“跟牌合法性”和“本墩赢家”。

## 默认配置

- 主花色：`♦`
- 级牌：`2`

## 固定首发牌型

南家（玩家0）首发 8 张：

- `♠A`
- `♠K`
- `♠Q ♠Q ♠J ♠J`（1 个拖拉机）
- `♠9 ♠9`（1 个对子）

合并写法：

`♠A ♠K ♠Q ♠Q ♠J ♠J ♠9 ♠9`

系统应按以下子结构理解：

1. 单牌 `♠A`
2. 单牌 `♠K`
3. 拖拉机 `♠Q ♠Q ♠J ♠J`
4. 对子 `♠9 ♠9`

## 用例列表

### 1. CUT_THROW_OK_01

类型：毙主成功

- 南：`♠A ♠K ♠Q ♠Q ♠J ♠J ♠9 ♠9`
- 东：`♥A ♥K ♥Q ♥Q ♥J ♥J ♥9 ♥9`
- 北：`大王 小王 ♦A ♦A ♦K ♦K ♦5 ♦5`
- 西：`♣A ♣K ♣Q ♣Q ♣J ♣J ♣9 ♣9`

预期：

- 东、北、西跟牌均合法
- 北家获胜

说明：

- 北家完全断黑桃，整手全主
- 北家形成了完整的“2单牌 + 1拖拉机 + 1对子”主牌结构
- 应视为有效整手毙牌

### 2. CUT_THROW_OK_02

类型：毙主成功（后位更大主牌再反超）

- 南：`♠A ♠K ♠Q ♠Q ♠J ♠J ♠9 ♠9`
- 东：`小王 ♦10 ♦10 ♦9 ♦9 ♦6 ♦5 ♦4`
- 北：`♥A ♥K ♥Q ♥Q ♥J ♥J ♥9 ♥9`
- 西：`大王 ♦A ♦A ♦K ♦K ♦J ♦8 ♦7`

预期：

- 东、北、西跟牌均合法
- 西家获胜

说明：

- 东家虽全主，但结构拆解后强度不应高于西家
- 西家后位用更强主牌结构反超，应最终获胜

### 3. CUT_THROW_NG_01

类型：毙主失败（部分同门 + 主牌补缺）

- 南：`♠A ♠K ♠Q ♠Q ♠J ♠J ♠9 ♠9`
- 东：`♠8 ♦A ♠10 ♠10 ♦K ♦K ♠3 ♠3`
- 北：`♥A ♥K ♥Q ♥Q ♥J ♥J ♥9 ♥9`
- 西：`♣A ♣K ♣Q ♣Q ♣J ♣J ♣9 ♣9`

预期：

- 东、北、西跟牌均合法
- 南家获胜

说明：

- 东家手里有部分黑桃，因此必须先跟尽黑桃
- 东家这手是“部分跟同门 + 主牌补缺”
- 即使跟牌合法，也不能把整手判成有效毙牌

### 4. CUT_THROW_NG_02

类型：毙主失败（全主但结构不完整）

- 南：`♠A ♠K ♠Q ♠Q ♠J ♠J ♠9 ♠9`
- 东：`大王 ♦A ♦K ♦K ♦10 ♦9 ♦8 ♦7`
- 北：`♥A ♥K ♥Q ♥Q ♥J ♥J ♥9 ♥9`
- 西：`♣A ♣K ♣Q ♣Q ♣J ♣J ♣9 ♣9`

预期：

- 东、北、西跟牌均合法
- 南家获胜

说明：

- 东家虽然完全断黑桃且整手全主
- 但未形成与首发对应的完整主牌结构，不能有效压过首发
- “全主”不等于“有效毙牌”

## 验证重点

1. 非主首引时，只有“完全断门后的整手主牌对应压制”才可改变赢家
2. “部分同门 + 主牌补缺”只能是合法跟牌，不能构成整手有效压制
3. “全主但结构不完整”也不能构成整手有效压制
4. 多家都使用主牌时，最终赢家应按有效主牌结构强度继续比较
