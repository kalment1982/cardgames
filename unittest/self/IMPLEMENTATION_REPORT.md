- **测试：** 6个测试全部通过
- **流程：** 完整的游戏流程，支持25墩出牌

### Task 3.5: 升级判定 (LevelManager)
- **功能：** 根据得分判定升级、确定下一局坐庄玩家
- **测试：** 5个测试全部通过
- **规则：**
  - 闲家<40分：庄家升3级
  - 闲家40-79分：庄家升2级
  - 闲家80-119分：闲家升1级
  - 闲家120-159分：闲家升2级
  - 闲家≥160分：闲家升3级
